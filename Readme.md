# Shop (An E-commerce)
An e-commerce made with RestFramework and ReactJs in the frontend. 

## Functional requirements
- Create account
- Login
- Logout
- Search product
- Add to / Delete from wishlist 
- Add to / Delete from cart 
- Checkout
- Reduce inventory on sale
- List orders
- Last seen
